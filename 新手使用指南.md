## ❓ 常见问题

### Q1: 为什么不能交易？

**A**: 当前系统只是在采集行情数据，还没有启用交易功能。这是正常的，因为：
1. 需要先积累足够的历史数据
2. 需要开发和测试交易策略
3. 需要配置 API Key 才能真实交易

### Q2: 数据会一直采集吗？

**A**: 是的，只要服务器运行，就会持续采集 OKX 的实时行情数据。

### Q3: 如何停止服务？

**A**: 如果需要停止，可以 SSH 到服务器执行：
```bash
ssh -i "C:\Users\11915\Desktop\蓝洞科技\mawentao.pem" ubuntu@43.198.18.252
cd TTQuant
docker compose -f docker/docker-compose.yml down
```

### Q4: 数据存在哪里？

**A**: 所有数据存储在 TimescaleDB 数据库中，位于 EC2 服务器的 Docker 卷中，会自动持久化。
